# Site Images

All images are stored locally in this directory (`public/images/`) and referenced directly by the components.

## Required Files

| File | Used For | Description |
|------|----------|-------------|
| `hero-bg.jpg` | Hero background | Plumbing tools graphic with water splashes |
| `service-drain.jpg` | Drain Cleaning card | Technician using drain snake machine at manhole |
| `service-plumbing.jpg` | General Plumbing card | Plumber servicing chrome P-trap piping under sink |
| `service-sewer.jpg` | Sewer Services card | Large green suction hose at sewer manhole |
| `service-emergency.jpg` | 24/7 Emergency card | White service van in motion on road |

## How to Add Images

Since binary files cannot be created via text output, save each attached image into this folder with the exact filename listed above. The components (`Hero.jsx` and `Services.jsx`) are already configured to reference these paths.
